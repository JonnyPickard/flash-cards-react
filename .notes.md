FlashCards
==========

Component design:
- Add a card
  - General
  - Code
- Cards List 
  - Filter
    - All
    - Known
    - Unknown
    - General
    - Code
- Card Show
  - Code
  - General

Model design:
- Cards
  - Id
  - Content
  - UnderstoodStatus
    - Known / Unknown
  - Type
    - Code / General
    
API design:
- Cards
  POST /cards
    > Return all cards 
      [ {card}, {card} ]
  GET /cards/filter/:filtertype
    > Return cards filtered by :filterype 
      [ {card}, {card} ]
  GET /cards/:id
    > Return particular card 
      {card}
  DELETE /cards/:id 
    > Delete particular card
